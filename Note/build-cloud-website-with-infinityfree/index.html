Build cloud website with Infinityfree

Posted on 2023-01-02 In Note Disqus:

PHP Cloud Hosting 雲端架站

一般如果你使用本地端的xampp進行架站，需要自行去申請網域，

並且主機要一直開著，比較麻煩

InfinityFree 是一個免費的虛擬主機服務，內建 PHP、MySQL，並提供免費的三個hosting account

下面分三個階段來介紹 一一 建立主機、建立資料庫、上傳程式碼檔案

註冊帳號

Infinityfree網址

點選上述連結進入網站後

點sign up now先進行註冊

註冊完之後會來到client area的頁面 (如果沒有的話，從首頁右上方的client area進入 )

建立主機

點create account
第一步是命名你的subdomain和選你的後綴網域(這邊就是在設定你的網站網址)

如果你有自己的網域，當然也可以選擇custom domain，要再去NameSilo做設定

不過這邊不多做討論，有需要的人可以參考此網址


再來是帳號密碼的部分

帳號都是由系統自動產生，密碼可以系統產生或是自行設定

完成！

接下來先到client area地方，這邊可以查看你的帳戶詳細的資訊

label 的資訊裡面是你網站的網址

下面是MySQL的相關資訊 包含username/password/hostname

再來介紹上面按鈕的功能

基本上就是使用前面這兩個 - control panel和file manager

點control panel會進入到後台介面，file manager是放檔案,程式碼的地方

建立資料庫

現在先建立我們的資料庫，所以先點選control panel進入到後台

看到下方有一個MySQL databases

從這邊去建立我們的資料庫，

一樣先命名資料庫，

底下可以看到已經建立的資料庫有哪些

上面也會顯示你的database name/username/hostname

(login.php檔案會需要這些資料)

password要去前面提到的client area頁面看

再來是建立你的資料表，點選Admin進入

進來到phpmyadmin頁面，在這邊建立你的table/data

上傳程式碼檔案

再來就是寫好的php檔案要放在哪裡

先回到client area的頁面點選file manager

下面是檔案管理的畫面，

你撰寫的php檔案或是其他檔案，一定要放進file manager裡面的htdocs資料夾

file manager裡面可以upload事先寫好的檔案

或是直接建立一個新的，直接在上面的簡易編輯器coding

若要對現有的檔案做修改，點一下就可以線上編輯，非常方便

如果前面資料庫建立好，檔案也都有放到htdocs資料夾，

以及php連線資訊修改成雲端上的資訊(就是login.php)

在瀏覽器打上自己的網址，可以看到網站已經在雲端上架設完成！！ 🦦
