Build blog with hexo

Posted on 2022-12-19 Edited on 2022-12-26 In Tutorial Disqus:

用光速建置基本版 Blog

Environment

工欲善其事，必先利其器，請安裝

  • Node.js ( 開啟命令提示字元 -> 輸入 node -v 檢查是否安裝完成 )
  • Git
  • Visual Studio Code
  • Hexo: 開啟命令提示字元 -> 輸入 npm install -g hexo-cli

Build

開啟命令提示字元，依序打入指令，即可無痛獲得 Blog 專案

  • hexo init YOUR_HEXO_FOLDER_NAME
  • cd YOUR_HEXO_FOLDER_NAME
  • npm install
  • hexo server // 為啟動專案指令，打開瀏覽器輸入 localhost:4000，Blog畫面近在眼前

New Post

  • hexo new post mynewpost
  • 打開 hexo 專案，相對路徑參考
    path:\YOUR_HEXO_FOLDER_NAME\source\_posts\mynewpost.md
  • 虛線底下輸入 # HI, MY FIRST POST 儲存並啟動專案
  • 恭喜你有一篇新文章，離部落客又更近了一點

Hexo Command

  • hexo new post POST_NAME (建立一個新的文章)
  • hexo new page PAGE_NAME (建立一個新的頁面)
  • hexo cl / hexo clean (清除快取檔案，如 db.json, pulic folder)
  • hexo g / hexo generate (產生靜態檔案)
  • hexo d / hexo deploy (部署到 Github)
  • hexo server (本地端啟動專案)

你可以探索的幾件事：

  • 更換主題 Hexo theme
  • 留言板功能 Hexo Disqus
  • 部署至Github Hexo deploy
  • 網站流量 Google analysis
  • Give me a cup of cafe 功能
